# Vehicle-Builder

## Description
Create a vehicle using the CLI from user input starting it using 'npm start' and you may begin creating an existing or new vehicle

## Table of Contents 
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## Installation

Install the application by cloning the repository to your computer.
You will need node.js and inquirer installed on your computer. to install inquirer use 'npm install inquirer'

## Usage

Open the terminal and go to where you installed the application.
use 'npm start' to start the application and begin creating your vehicle by following the CLI
